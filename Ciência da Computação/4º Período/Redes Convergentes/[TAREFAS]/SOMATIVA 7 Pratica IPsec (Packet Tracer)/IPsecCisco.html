Prática de IPsec com CISCO Packet Tracer


Roteadores utilizados: 2811

CENÁRIO:




INSTRUÇÕES:

O cenário inicial já está fornecido em um arquivo pkt (Cisco Packet Tracer ) no AVA.

Caso queria criar o cenário sem usar o arquivo pronto, escolha as portas dos roteadores conforme abaixo:

(Fa0/0)  RA  (Fa0/1) --- (Fa0/0)  R  (Fa0/1) --- (Fa0/1)  RB  (Fa0/0)

Aplique os scripts mostrados abaixo nos roteadores. Os PCs já estão configurados.

Para aplicar o script:

Clique no Roteador onde irá aplicar o SCRIPT.
Digite <ENTER>
Faça um copy-and-paste do SCRIPT.


Script: RouterA (PARIS)

 
!ROTEADOR PARIS
enable
configure terminal
hostname PARIS
!
!DEFINE A FORMA DE AUTENTICACAO
crypto isakmp policy 1
encr aes
authentication pre-share
group 2
exit
crypto isakmp key TESTE address 10.0.0.2
!
!DEFINE A FORMA DE PROTECAO DOS DADOS (TRANSFORMACAO)
crypto ipsec transform-set SECUREWAN esp-aes esp-sha-hmac
crypto map IPSECWAN 100 ipsec-isakmp
set peer 10.0.0.2
set pfs group2
set security-association lifetime seconds 86400
set transform-set SECUREWAN
match address 100
exit
!
!DEFINE O TRAFEGO QUE SERA ENVIADO PARA VPN
access-list 100 permit ip 10.1.1.0 0.0.0.255 172.16.2.0 0.0.0.255
!
!DEFINE A INTERFACE QUE FARA O TUNELAMENTO
interface FastEthernet0/0
ip address 10.1.1.2 255.255.255.0
no shutdown
exit
interface FastEthernet0/1
ip address 172.16.1.1 255.255.255.0
crypto map IPSECWAN
no shutdown
exit
ip route 0.0.0.0 0.0.0.0 172.16.1.2
end
