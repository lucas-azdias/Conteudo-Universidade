A) REVISÃO DE CONCEITOS

Um mesmo número pode ser representado em BASES diferentes, mas fisicamente, os dispositivos computacionais manipulam números APENAS em binário.

A representação em uma  BASE é apenas uma forma de VISUALIZAR O NÚMERO, ela não muda seu valor.

BASE é o número de digitos usados para representar o número.

Os dígitos vão de 0 até (BASE - 1). Por exemplo, a base decimal usa digitos de 0 até 9

A base 16 usa dígitos de 0 até 15. Os dígitos 10 em diante são substituídos pelas letras A até F.


# representa o número decimal em binário (base 2 - digitos 0 até 1)
print("Numero 192 (decimal => binario) = ", format(192,'8b')) 

# representa o número decimal em hexadecimal (base 16 - digitos de 0 até 15)
print("Numero 192 (decimal => hexa) = ",format(192, '1x'))

# um digito em hexa corresponde a 4 bits
print("Digito c (hexa => binario) = ", format(0xc, '4b')) 

# um digito em hexa corresponde a 4 bits
print("Digito f (hexa => binario) = ", format(0xf, '4b'))


Obs. Na babilônia, a representação de números usava a base 60 (sexagesimal)!!!

B) O TAMANHO DA PALAVRA

Computadores não fazem apenas operações em um bit de cada vez.

Bits são agrupados em palavras de tamanho variável, para serem manipulados em uma única operação (instrução do microprocessador).

A quantidade de bits usado em uma palavra depende do tamanho do número que se deseja representar.


# representa o número decimal 192 com um byte (palavra de 8 bits)
print("Numero 192 (8 bits) = ", format(192,'b').zfill(8)) 
print("De : ", 0b00000000, " até ", 0b11111111)
print("De : ", 0x00, " até ", 0xff)

# representa o número decimal 192 com um 2 bytes (palavra de 16 bits)
print("Numero 192 (16 bits) = ", format(192,'b').zfill(16)) 
print("De : ", 0b0000000000000000, " até ", 0b1111111111111111)
print("De : ", 0x0000, " até ", 0xffff)

# o maior número que pode ser representado é 2 elevado (TAMANHO DA PALAVRA - 1 )
print("8 bits: ", 2**8)
print("16 bits: ", 2**16, ' ou ', (2**8)*(2**8), ' ou ', (2 ** 8)**2)
print("32 bits: ", 2**32, ' ou ', ' ou ', (2 ** 8)**4)
print("64 bits: ", 2**64, ' ou ', ' ou ', (2 ** 8)**8)

C) TEXTO vs BINÁRIO (Não Texto)

Um DÍGITO é uma representação de um número

Um CARACTERE é uma representação de uma letra (ou símbolo). Um texto é formado por caracteres.

Observe o caractere '0' é diferente do número 0. Os sistemas usam " " ou ' ' para identificar caracteres.

Computadores representam qualquer tipo de informação como números. No caso de caracteres, esses números são interpretados como CÓDIGOS.

Para interpretar algo como texto é necessário DIZER que os números em binário DEVEM SER TRADUZIDOS por uma TABELA DE CODIFICAÇÃO.

Quando a codificação não é fornecida de forma explícita, muitos sistemas assumem que o texto está codificado em ASCII ou UTF-8 (que é o caso do Python).

def ror(n, chave, tam):
    return (2**tam-1) & (n >> chave | n << (tam-chave) )

def rol(n, chave, tam):
    return (2**tam-1) & (n << chave | n >> (tam-chave) )

def cripto(mensagem, chave):
    cripto = ""
    for c in mensagem:
        codigo = ord(c) + chave
        cripto += chr(codigo)
    return cripto

plain = 'Nada de novo no front'
print(cripto(plain, 3))
cipher = cripto(plain, 4)
print(cipher)
print(cripto(cipher,-4))
